# 🦸‍♂️ Hero App - Full Stack Auth App (React + Express + Prisma + JWT + Postgres)

This is a full-stack authentication app with:

- **React** Frontend
- **Express.js** Backend
- **Prisma ORM** with **PostgreSQL**
- **JWT Authentication**
- Login / Signup flow ✅
- Protect Home Page ✅

## 🚀 Tech Stack
- Frontend: React
- Backend: Express.js
- Database: PostgreSQL
- ORM: Prisma
- Authentication: JWT
